Qu'est-ce que c'est?
---------------

Ce projet est un exercice qui a pour but de savoir utiliser Git et Github.
Il entre dans le cadre de ma formation au module :Gérez votre code avec Git et GitHub, de la plateforme de formation OpenClassrooms.
Tous les elements à l'interieur sont fictifs.

----------------

- index.html
- style.css
- README.txt

- img


Modifications
-----------------

les modifications apportées sont faite sous forme de commit et téléversé dans le repertoire : Activite2_git_OC.
Pour les besoins de l'exercice 4 commits distincts seront ajouté afin de montrer l'ajout ou la modification de fichiers .
